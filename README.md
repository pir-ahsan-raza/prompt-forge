# prompt-forge

A curated collection of prompts, skills, and settings I've built up over time while working with AI tools - mostly Claude and GPT. Nothing academic here, just stuff that actually works in day-to-day developer life.

## What's inside

```
prompts/
  writing/     → tone, style, and content prompts
  code/        → dev-specific prompts (reviews, debugging, docs)
  research/    → summarization, analysis, comparison prompts
  productivity/→ planning, email, meeting prompts
skills/
  claude/      → Claude-specific skill definitions
  chatgpt/     → GPT system prompts and configs
settings/      → recommended model settings per use case
templates/     → reusable prompt templates with variables
```

## Philosophy

I got tired of copy-pasting half-working prompts from random Reddit threads and blog posts written by people who clearly don't use these tools daily. So I started keeping my own. The prompts in here are:

- Written like a human, not a robot trying to sound professional
- Tested on actual work - not toy examples
- Commented where the reasoning matters
- Updated when they stop working as models change

## How to use

Browse by folder based on what you're trying to do. Each prompt file has a quick description at the top, the prompt itself, and usually a note on what variations work well.

For skills (Claude-specific), drop them into your Project instructions or paste them at the start of a conversation.

## Contributing

If you've got something that works well, open a PR. Keep the same file format and be honest about limitations.
